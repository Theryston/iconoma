# Iconoma + Next.js Example

This example demonstrates how to use **Iconoma** in a [Next.js](https://nextjs.org) app.

## Getting Started

1. **Clone this repository**.

2. **Install dependencies:**

   ```bash
   cd examples/with-nextjs
   pnpm install
   ```

3. **Menage your icons:**

   Make sure to start the Iconoma studio:

   ```bash
   npx @iconoma/cli build
   ```

4. **Start the Next.js dev server:**

   ```bash
   npm run dev
   ```

5. **Opens the Next.js app**

   Opens [http://localhost:3000](http://localhost:3000) at your browser!
